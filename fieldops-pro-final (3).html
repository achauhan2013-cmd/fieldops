📧 Email Notifications (EmailJS)
EmailJS is free (200 emails/month) — no backend needed.

Setup steps:
1. Sign up at emailjs.com
2. Add an Email Service (connect Gmail)
3. Create one Email Template — set To Email to {{to_email}} and use any of these variables in the body:

{{engineer_name}}  ·  {{task_name}}  ·  {{task_status}}
{{task_location}}  ·  {{task_priority}}  ·  {{task_due}}
{{task_notes}}  ·  {{admin_email}}

4. Copy your Public Key (Account → General), Service ID, and Template ID into the fields below.

📬 Notifications sent automatically to:
 • Engineer — when admin assigns a new task to them
 • Admin — when engineer completes a task, changes status, or adds a comment
